Git Commands Reference

Basic Commands

git init

Initializes a new Git repository.

Usage: git init [repository-name]

git clone [url] [directory]

Clones a repository into a new directory.

Options: --branch <branch>, --depth <depth>

git add [file]

Adds changes to the staging area.

Options: ., -A, -p

git commit -m "commit message"

Records changes to the repository.

Options: -a, --amend

git status

Shows the working tree status.

git log

Shows the commit history.

Options: --oneline, --graph, --author=<author>

git diff [options]

Shows changes between commits, commit and working tree, etc.

Options: HEAD, --cached

git branch [branch-name]
Copy

Lists, creates, or deletes branches.

Options: -d, -m

git checkout [branch-name]

Switches branches or restores working tree files.

Options: -b, --

git merge [branch-name]

Merges branches.

Options: --no-ff

git pull [remote] [branch]

Fetches from a remote repository and merges.

Options: --rebase

git push [remote] [branch]

Updates remote refs along with associated objects.

Options: --force, --tags

git remote [options]

Manages set of tracked repositories.

Options: add, remove, -v

Advanced Commands

git rebase [branch]

Reapplies commits on top of another base tip.

Options: -i

git cherry-pick commit

Apply the changes introduced by some existing commits.

Options: -n

git stash

Stashes changes in a dirty working directory.

Options: apply, pop, list

git tag [tag-name]

Creates, lists, or deletes tags.

Options: -a, -d

git fetch [remote]

Downloads objects and refs from another repository.

Options: --prune

Common Workflows

mkdir my-repo && cd my-repo && git init

Create a new Git repository.

git remote add origin remote-repo-url

Add a remote repository (e.g., on GitHub).

git branch branch-name

Create a new branch.

git checkout branch-name

Switch to the new branch.

git branch -d branch-name

Delete a branch.

git add

Stage changes.

git commit -m "Your commit message"

Commit changes.

git clone repository-url

Clone a repository to your local machine.

git tag tag-name

Create a tag.

git push origin branch-name

Push changes to the remote repository.

git pull origin branch-name

Pull changes from the remote repository.

git remote -v

View the remote repositories you have configured.

git remote remove remote-name

Remove a remote repository.

Viewing Existing Elements

git remote -v

View remote repositories.

git branch

Lists local branches.

git branch -r

Lists remote branches.

git branch -a

Lists all branches (local + remote).

git tag

Lists all tags.

git log

Shows commit history.

Creating New Elements

git remote add remote-name remote-url

Create a new remote.

git branch new-branch-name

Create a new branch.

git tag new-tag-name

Create a new tag.

git add file

Create a new commit.

git commit -m "Your commit message"

Commit your changes.

Deleting Elements

git remote remove remote-name

Delete a remote.

git branch -d branch-name

Delete a branch.

git tag -d tag-name

Delete a tag.

git reset --soft HEAD~1

Remove the last commit (keep changes).

git reset --hard HEAD~1

Remove the last commit (discard changes).

Renaming Elements

git remote rename old-remote-name new-remote-name

Rename a remote.

git branch -m new-branch-name

Rename a branch (if on the branch).

git branch -m old-branch-name new-branch-name

Rename a branch (if on a different branch).

git tag -d old-tag-name

Delete the old tag.

git tag new-tag-name

Create a new tag with the new name.
